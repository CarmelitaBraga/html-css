HTML: Main formatting types

Highlights

You should not use the obsolete and not semantic tag <b> to highlight some important aspect of your text.

Try using this <strong> instead, which is full of meaning in a html context.

Emphasized

You can put emphasis to your text with the <em> tag, full of meaning.

The archaic one using the <i>, which is only form, no further meaning.

Other marking types

Markers

You can also mark up some aspect of your text that's very important with the <mark> tag.

This other important text will have different colors, the standard color.

Small case

You can write small case texts with the <small> tag.

Deleted text

And also can delete some aspect of your text that you dislike using the <del> tag.

Avoid using the <s> tag.

Inserted text

Let's say you'll add up some extra text, use the <ins> tag.

The <u> tag is empty of meaning, avoid using it

Subscript text

Let's write down some chemical formula such as H3PO4.

Use the <sub> tag

Superscript text

Let's write down some mathematical formula using power, such as x3 + 5x2

Use the <sup> tag
